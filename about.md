# 关于 Python 训练项目中心

这是一个专注于 **Python 实战项目** 的学习平台，涵盖从基础语法到机器学习的完整学习路径。

## 项目分类

| 分类 | 难度 | 适合人群 |
|------|------|----------|
| 基础语法 | ⭐ 入门 | Python 零基础学员 |
| 数据分析 | ⭐⭐ 初级 | 想掌握数据处理技能的开发者 |
| Web 开发 | ⭐⭐⭐ 中级 | 希望学习后端开发的学员 |
| 自动化脚本 | ⭐⭐ 初级 | 想提升工作效率的办公人员 |
| 机器学习 | ⭐⭐⭐⭐ 进阶 | 有基础想深入 AI 领域的开发者 |

## 技术栈

- **内容管理**：VitePress（Markdown）
- **部署平台**：Cloudflare Pages
- **代码示例**：Python 3.x

## 学习建议

1. 按顺序完成每个项目，不要跳过基础
2. 亲自敲一遍代码，不要只看不练
3. 尝试修改代码，观察不同结果
4. 完成项目后，尝试独立实现类似功能
