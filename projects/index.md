# Python 训练项目

欢迎来到 Python 训练项目中心！这里按难度和领域分类整理了各类实战项目。

## 项目分类

| 分类 | 难度 | 描述 |
|------|------|------|
| [基础语法](/projects/basics/) | ⭐ 入门 | 变量、循环、函数、面向对象 |
| [数据分析](/projects/data-analysis/) | ⭐⭐ 初级 | Pandas、NumPy、可视化 |
| [Web 开发](/projects/web/) | ⭐⭐⭐ 中级 | Flask、FastAPI 框架 |
| [自动化脚本](/projects/automation/) | ⭐⭐ 初级 | 爬虫、办公自动化 |
| [机器学习](/projects/ml/) | ⭐⭐⭐⭐ 进阶 | Scikit-learn、经典算法 |

## 学习路径建议

1. **零基础**：从 [基础语法](/projects/basics/) 开始，掌握 Python 核心概念
2. **有基础**：选择 [数据分析](/projects/data-analysis/) 或 [自动化脚本](/projects/automation/) 练手
3. **进阶提升**：挑战 [Web 开发](/projects/web/) 和 [机器学习](/projects/ml/)
